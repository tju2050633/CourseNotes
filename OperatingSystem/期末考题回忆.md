```c
semaphore mutex = 1;
semaphore empty = 100;

Reader
{
  while(1)
  {
    P(mutex);
    P(empty);
    V(mutex);
    阅读;
    P(mutex);
    V(empty);
    V(mutex);
  }
}
```

### 3.分页，给页表，逻辑地址转物理地址

（很怪的题，但老师反复说没出错）

| 页号             | 块号   | 是否在内存 |

| 0                | 3      | 1          |
| 1                | 用不到 | 1          |
| 2                | 用不到 | 1          |
| 。。。（用不到） |        | 0          |



- 逻辑地址2148（十进制）的物理地址？
  - 3\*1024\*1024+2148
- 逻辑地址5000的物理地址？
  - 3\*1024\*1024+5000
